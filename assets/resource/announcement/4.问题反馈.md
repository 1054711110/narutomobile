# 问题反馈

1. 遇到问题先重启电脑，不行就重新下载并解压最新版 [Github](https://github.com/duorua/narutomobile/releases) [Mirror酱](https://mirrorchyan.com/zh/projects?rid=MaaAutoNaruto&source=man-announce) [QQ群](https://qm.qq.com/q/Mck36YvBsu)

2. 确定不是您的个人问题，选择上报方式 [Github](https://github.com/duorua/narutomobile/issues) [QQ群](https://qm.qq.com/q/Mck36YvBsu)

3. 准备好您的**上报材料**，开发者不是情报自来也，不能靠算卦来判断出错误的原因。请[导出日志](https://naruto.natsuu.top/guide/issue.html)并尽可能提供详细的信息，能方便开发者快速定位问题。

4. 如果无法打开MAN或[导出日志](https://naruto.natsuu.top/guide/issue.html)失败，请打包提交MAN 文件夹下名叫 `logs` 和 `debug` 的两个文件夹，除此之外您还可以使用录屏软件录制运行出错的过程。

5. (Q群)如果您选择在Q群上报，请 @管理员
