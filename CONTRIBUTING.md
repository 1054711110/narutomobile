## 重要事项

您通过任意方式提交代码到 **MaaAutoNaruto** 下属任意仓库，即代表您理解并同意以下条款：

- 您授权 **MaaAutoNaruto** 的开发团队 **MaaAutoNaruto-DevGroup** 使用您所提交代码的部分权利：

&nbsp;   1. **版权许可**  

&nbsp;   您授予 **MaaAutoNaruto-DevGroup** 一项永久的、全球性的、不可撤销的、免版税的、非独占的版权许可，允许其使用、复制、修改、分发、公开演示和展示您的贡献，并基于您的贡献创作衍生作品。

&nbsp;   2. **专利许可**  

&nbsp;   您同时授予 **MaaAutoNaruto-DevGroup** 一项永久的、全球性的、不可撤销的、免版税的专利许可，覆盖您贡献中所必需的专利权利。

&nbsp;   3. **保留权利**  

&nbsp;   您仍保有您原创贡献的版权和署名权。您的姓名将作为作者保留在 Git 提交历史及项目致谢中。

- **MaaAutoNaruto-DevGroup** 的所有权利均授权给 [duorua (@duorua)](https://github.com/duorua)，仅该被授权人能代表 **MaaAutoNaruto-DevGroup** 行使全部权利。补充细则如下：

1. 无法联系到 [duorua (@duorua)](https://github.com/duorua) 时，**MaaAutoNaruto-DevGroup** 相关事务由 [MaaAutoNaruto 主要开发者](https://github.com/MaaAutoNaruto-DevGroup) 依据社区共识处理。

2. [duorua (@duorua)](https://github.com/duorua) 退出项目时，必须将 **MaaAutoNaruto-DevGroup** 的所有授权移交给另一项目开发者。

3. [MaaAutoNaruto 主要开发者](https://github.com/MaaAutoNaruto-DevGroup) 绝对多数通过时，允许取消对 [duorua (@duorua)](https://github.com/duorua) 的授权，由 [MaaAutoNaruto 主要开发者](https://github.com/MaaAutoNaruto-DevGroup) 重新授权给其他项目开发者。

以上条款用于保证 **MaaAutoNaruto** 项目拥有清晰的责任主体，能够对侵权者采取法律手段进行法律维权，并确保 **MaaAutoNaruto** 的权利不会被滥用。
